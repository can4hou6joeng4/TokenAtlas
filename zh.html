TokenAtlas

为你的本地 AI 编码用量绘制地图 —— 令牌、成本、会话与 Git 活动 —— 集于一个安静的原生 macOS 菜单栏 App。它直接读取 Claude Code 与 Codex 已写入磁盘的痕迹。无需 API key、无需账号、无遥测。

00 · 概览

你的工作,直接从磁盘读取。

默认安静,需要时密集 —— 一个原生工具,而非营销仪表盘。

01 · 功能

关于编码日常的一切,尽在菜单栏。

  1. 菜单栏用量地图始终一键可达

    令牌、预估成本、缓存活动、最近会话与提供方状态 —— 从菜单栏一览无余。

  2. 多提供方设计当前支持 Claude Code 与 Codex

    提供方特有的细节藏在干净的适配器协议之后,新的 CLI 无需改动共享代码即可接入。

  3. 会话与项目完全在本地

    查看对话、项目、消息、模型构成与个人记录 —— 不向任何托管服务发送数据。

  4. 仓库活动用量映射到提交

    将 AI 编码用量与本地 Git 历史、内置的各语言统计相关联。

  5. 可选的灵动岛一瞥即得的面板

    基于 Atoll 的面板,展示活动、统计、媒体、计时、剪贴板等 —— 想用时再开。

  6. 打包式自动更新Sparkle

    手动检查与静默后台更新,经签名的 Sparkle appcast 分发。

02 · 原则

几件绝不妥协的事。

  1. 1

    本地优先

    统计读取自 ~/.claude、~/.codex 等本地工具数据。你的数据从不离开你的 Mac。

  2. 2

    默认安静

    克制的配色、稳定的表格、易读的数字 —— 一个原生 macOS 工具,而非喧哗的仪表盘。

  3. 3

    多提供方

    一张地图,容纳每个 CLI。新增提供方 = 一个文件夹、一次协议实现、一行注册。
